# Design System

## Visual language

- Editorial enterprise look with warm neutral surfaces and deep navy text
- No generic dark SaaS dashboard aesthetic

## Typography

- Display: Syne
- Body: Manrope

## Tokens

- `--bg-main`, `--bg-panel`, `--bg-ink`
- `--text-primary`, `--text-muted`
- `--accent-teal`, `--accent-amber`, `--accent-rose`

## Accessibility

- WCAG AA color contrast targets
- Keyboard-first interaction for all key actions
- Explicit focus rings and reduced-motion support
